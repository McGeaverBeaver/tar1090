Historical import — globe_history heatmap → searchable index
Scans readsb's heatmap .ttf chunks under this path and fills the flight index, so traffic from before logging began (or with traces off) becomes searchable. Runs in the background and resumes automatically if it crashes or the container restarts. Imported flights have no callsign (the heatmap doesn't store it) — search them by registration / type / operator / military / time.
